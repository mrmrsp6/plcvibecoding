# 公开功能清单

本清单仅包含用户可在桌面应用中直接感知到的功能，不包含内部实现细节。

## 1. 工程导入与品牌支持

| 功能 | 说明 |
|------|------|
| 三菱 GX Works 工程导入 | 支持 `.gxw` 等 GX Works 工程文件导入 |
| 西门子 TIA 工程导入 | 支持 `.ap16` 等 TIA Portal 工程文件导入 |
| 虚拟工程创建 | 不依赖真实 PLC 或官方软件，即可创建最小工程进行原型验证 |
| 双品牌工作区 | 同一界面下切换三菱/西门子品牌视图 |

## 2. 连续梯形图工作区

- 以连续滚动方式展示梯形图，替代传统分页浏览。
- 支持主程序、子程序、功能块（FB）之间的跳转与高亮。
- 节点、梯级、支路可点击联动。

## 3. AI 协作面板

- 普通对话模式：自由询问 PLC 相关知识。
- PLC 专家模式：基于当前工程上下文给出分析、审查与修改建议。
- 支持多种大模型后端：OpenAI、Anthropic、DeepSeek、Ollama 等。

## 4. Patch 预览与安全边界

- 所有 AI 生成的修改先以补丁形式呈现。
- 工程师可预览修改前后的梯形图差异。
- 写回工程或 PLC 前需要明确授权，避免误操作。

## 5. Prompt 护栏与技能召回

- 内置通用 PLC 协作护栏（见 [`skills/SKILL.md`](../skills/SKILL.md)）。
- 根据用户输入自动召回相关的安全规则与检查清单。

## 6. 会话与项目级缓存

- 项目级对话流加密缓存，保护工程敏感信息。
- 切换工程时自动隔离会话上下文。

## 7. 独立打包与发布

- Electron 桌面应用，支持 Windows 安装包（NSIS）与便携版。
- 安装包在 [GitHub Releases](https://github.com/mrmrsp6/plcvibecoding/releases) 中发布。

## 不在本仓库中的内容

- 桌面应用源码
- 离线工程解析实现
- 内部路由与分析框架
- 品牌私有协议适配细节
