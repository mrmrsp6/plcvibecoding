# 三菱结构原语示例

这是一个已脱敏的最小三菱 GX Works 示例工程，用于演示三菱 Q 系列工程的基本结构原语。

## 文件

- `payload/public-structure-primitives.gxw`：示例工程文件，可在 GX Works 中打开。
- `redaction-report.md`：脱敏声明与校验信息。

## 覆盖的梯形图元素

- 基本 LD / OUT 逻辑
- 内部继电器 M 的使用
- 简单程序结构组织

## 使用方式

1. 在 GX Works 中打开 `payload/public-structure-primitives.gxw`。
2. 结合本仓库 [`skills/SKILL.md`](../../../skills/SKILL.md) 的护栏，让 AI 协助解释基础结构。
3. 任何修改建议先在离线环境或仿真器中验证。
